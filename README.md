# 🌐 Unique Landing Page

A modern and responsive landing page built using **HTML, CSS, and JavaScript**. This project features an animated hero section, glowing text effects, floating background shapes, smooth scrolling navigation, and a clean user interface.

## ✨ Features

* 🎨 Modern and attractive UI design
* 🌈 Gradient glowing text effects
* 🫧 Floating animated background shapes
* 📱 Fully responsive layout
* 🚀 Smooth scrolling navigation
* 🎯 Interactive "Explore Now" button
* 📌 Sticky navigation bar with scroll effect
* ⚡ Built using pure HTML, CSS, and JavaScript

## 📂 Project Structure

```
Unique-Landing-Page/
│── index.html      # Main HTML file
│── README.md       # Project documentation
```

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/unique-landing-page.git
```

2. Navigate to the project folder:

```bash
cd unique-landing-page
```

3. Open `index.html` in your preferred web browser.

No additional setup or dependencies are required.
